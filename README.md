# 3D Capybara Music Player
#### An interactive 3D-printed capybara that uses an RFID reader to identify 3D-printed bug tokens. Each RFID-tagged bug triggers a Spotify playlist to play when placed on the capybara's head.

#### RFID or Radio Frequency IDentification utilizes radio waves in order to read RFID tags commonly found in warehouses and factories for asset management. This project utilizes RFID readers and tags, with the help of a Raspberry Pi running a python program which switches the song based on the value it reads from the scanner. 

## Components
#### Hardware
    * Raspberry Pi Zero 2 W
    * RFID Reader RC522
    * RFID Stickers (13.56MHz)
    * Speaker
    * Raspberry Pi Zero 2 W Power Cable
    * Dupont Cables
#### Software
    * Raspberry Pi OS (Bullseye)
    * Thony Python IDE
    * Spotify Web API
#### Design
    * Ender Pro 3D Printer
    * Blender
    * Ultimaker Cura
    * Procreate
